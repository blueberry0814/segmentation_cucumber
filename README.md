오이만 segmentation

- **`default` or `vit_h`: [ViT-H SAM model.](https://dl.fbaipublicfiles.com/segment_anything/sam_vit_h_4b8939.pth)**

2. requirements.txt 를 참고해 환경 세팅
3. final.py 실행
